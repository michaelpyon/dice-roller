# High Roller

A dice rolling toy with streak multipliers and a casino aesthetic. Roll two dice, chase lucky 7s, doubles, and snake eyes. Each lucky roll stacks a streak multiplier with themed particle bursts.

## Features

- Two animated dice with shake transitions
- Three lucky outcomes: lucky 7 (or 11), doubles, snake eyes
- Streak multiplier that stacks while you keep hitting
- Particle burst celebrations themed per outcome
- Roll history (last 20 rolls)
- Keyboard support: Space or Enter to roll
- Mobile-friendly tap to roll

## Stack

- Vite 7
- React 19
- Tailwind CSS 4
- Vercel for deploys

## Develop

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Deploy

Linked to Vercel. Push to `main` and it auto-deploys, or run `npx vercel --prod`.
